This is a workload of 40 clients intended for performance analysis and profiling.  This is enough clients that network logic bottlenecks begin to become apparent.

rm -r world/cuboids world/entities && \
java -Xmx256m -jar october-server/target/october-server-0.0-SNAPSHOT-jar-with-dependencies.jar  9999
